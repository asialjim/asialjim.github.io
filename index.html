Remote

  • Remote is a declarative Java workflow processing framework. Based on this framework, any business process can be broken down into individual steps using annotations and a lifecycle approach. These steps are then assembled into a Java interface. Users only need to focus on the specific steps of the process without concerning themselves with the implementation of the interface.
